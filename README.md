# tic40.github.io

## site

https://tic40.github.io

## Install

```
$ brew install hugo
```


## Add new post

```
$ hugo new posts/{year}/{title}.md
```

## Run server on local


```
$ hugo server -D
```
*-D option is: include content marked as draft


## deploy

Just run the script!
```
$ ./deploy.sh
```
